despues iniciar la estancia con "git init"

despues agregar la conexion remota con git remote add origin (url de su repositorio)

luego si ya tienen archivos cargarlos con git add (nombre de o los archivos)

luego hacer tu primer commit con git commit -m "mensaje del commit"

luego hacer push con git push -u origin main
